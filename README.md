calc-cli

мини калькулятор на c#

запуск:
dotnet run

пример:
calc-cli add 2 3
calc-cli sub 10 5
calc-cli mul 3 4
calc-cli div 8 2
calc-cli pow 2 8

help:
calc-cli --help
или ввести --help в самой программе

делить на ноль ненано

команды:
add - сложить
sub - вычесть
mul - умножить
div - делить
pow - степень

для справки по dotnet:
dotnet --help
